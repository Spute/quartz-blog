## 功能

- <strong>多种内容支持</strong>

  - 视频、图集、音乐、直播信息下载
  - 支持个人主页、作品分享、直播、合集、音乐集合等多种链接
  - 支持去水印下载
- <strong>批量下载能力</strong>

  - 多线程并发下载，提升下载速度
- <strong>灵活配置</strong>

  - 支持命令行参数和配置文件两种方式，如：选择下载类型，下载时间段
  - 可自定义下载路径、线程数等
  - 支持下载数量限制
- <strong>增量更新</strong>

  - 自动跳过已经下载视频

## 步骤

- 下载项目代码。github 项目地址：[https://github.com/jiji262/douyin-downloader](https://github.com/jiji262/douyin-downloader)
- 安装 python3 解释器：这步可参考网上教程
- 安装项目运行的虚拟环境：`python3 -m venv venv`
- 进入虚拟环境：`source venv/bin/activate`
- 安装项目依赖的第三方包:  `pip install -r requirements.txt`
- 运行代码测试：`python3 DouYinCommand.py`
