# 豆瓣读书短评爬取

准备研究一下 AI 书单号，想着将真实的书籍评论都爬取下来做成一个知识库，对书单的文案生成肯定很有帮助。于是找到一篇教程，动手实践了一下，目前已经实现了短评的爬取，不到两分钟爬取了 460 条短评，效率杠杠的。

参考的教程链接：[https://www.bilibili.com/opus/654521185930838022](https://www.bilibili.com/opus/654521185930838022)

PS: 将这个教程代码中的 HTML 特殊字符，替换一下就能执行成功，麻烦一点的是替换自己的 cookies，不过按照教程也能顺利完成。

# 豆瓣读书长评爬取

继续昨天的豆瓣读书评论爬取工作。完成昨天未实现的长文评论爬取。长评相对于短评涵盖的信息更多，对书籍介绍文案的制作更有帮助。

依旧是参照 [https://www.bilibili.com/opus/654521185930838022](https://www.bilibili.com/opus/654521185930838022)（虽然这篇教程有些地方写的简单。但胜在不用自己分析页面结构，可以节省时间。）

处理好相关代码后，运行，发现报了 403 异常

应该是昨天的 cookie 过期了，重新打开网页获取 cookie

额。。。结果发现自己的 IP 被封了

![[K4fwbCNYnorNjTxUKHEc9M78n0e.png]]

还好我有其他办法，将电脑的 WiFi 改连到手机热点。这样爬虫程序的出口 IP 就变成另外一个了，这样就能正常访问页面，获取到 cookie 了。

当然也可以找个代理池，不过这样更麻烦。

看样子后面还得严格控制爬虫的频率了
