## 学习知识点

插件的内容非常简单，只有一个基础的 manifest.json 文件，外加一个 rules.json 文件，即实现了图片的替换。该规则会将所有网页中请求的 jpg 或 jpeg 格式图片重定向为指定的猫咪图片。

该插件基于 Chrome 的 declarativeNetRequest API，通过在 `rules.json` 文件中定义网络请求拦截规则。当网页请求 jpg 或 jpeg 格式的图片时，插件会自动将这些图片的请求重定向到指定的猫咪图片链接。所有规则在 manifest.json 中声明并启用，无需额外的脚本即可实现图片替换效果。

```json
[
  {
    "id": 1,
    "priority": 100,
    "action": {
      "type": "redirect",
      "redirect": {
        "url": "https://i.chzbgr.com/completestore/12/8/23/S__rxG9hIUK4sNuMdTIY9w2.jpg"
      }
    },
    "condition": { "urlFilter": "*.jpeg", "resourceType": ["image"] }
  },
  {
    "id": 2,
    "priority": 100,
    "action": {
      "type": "redirect",
      "redirect": {
        "url": "https://i.chzbgr.com/completestore/12/8/23/S__rxG9hIUK4sNuMdTIY9w2.jpg"
      }
    },
    "condition": { "urlFilter": "*.jpg", "resourceType": ["image"] }
  }
]
```
