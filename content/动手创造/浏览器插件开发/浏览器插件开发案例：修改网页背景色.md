# 学习知识点

## 项目结构

有选项页面、弹窗页面，已经两者对应的逻辑文件

```

README.md // 项目说明文档

images/  图标文件夹
```

## 核心代码

点击插件弹窗页面的颜色块，实现的页面的颜色变更。实现原则主要是弹窗页面 JS （popup.js 文件），如下是文件的具体实现过程：

```javascript
const changeColorButton = document.getElementById('changeColor');

// 从存储中获取颜色，并更新按钮的样式和属性
chrome.storage.sync.get('color', ({ color }) => {
  changeColorButton.style.backgroundColor = color;
  changeColorButton.setAttribute('value', color);
});

changeColorButton.addEventListener('click', (event) => {
  const color = event.target.value;

  // 注：在注入内容脚本前查询当前活动标签页
  chrome.tabs.query({ active: true, currentWindow: true }, (tabs) => {
    // 使用 Scripting API 执行脚本
    chrome.scripting.executeScript({
      target: { tabId: tabs[0].id },
      args: [color],
      func: setColor
    });
  });
});

function setColor(color) {
  // 将页面背景色设置为指定颜色
  document.body.style.backgroundColor = color;
}
```

选项页面逻辑

```javascript
const buttonDiv = document.getElementById('buttonDiv');
const buttonColors = ['#3aa757', '#e8453c', '#f9bb2d', '#4688f1'];

const createColorButtons = (buttonColors) => {
  buttonColors.forEach((color) => {
    const button = document.createElement('button');
    button.style.backgroundColor = color;

    button.addEventListener('click', () => {
      chrome.storage.sync.set({ color }, () => {
        console.log(`color is ${color}`);
      });
    });

    buttonDiv.appendChild(button);
  });
};

createColorButtons(buttonColors);
```
