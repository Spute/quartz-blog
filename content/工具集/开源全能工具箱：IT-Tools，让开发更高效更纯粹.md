## <strong>前言</strong>

在日常开发中，我们总是离不开各种在线小工具——格式化 JSON、生成 UUID、转换时间戳、加密解密、甚至临时调试 HTTP 请求。可这些功能往往分散在不同网站上，使用起来既繁琐又容易被广告打断。有没有一种方式，可以把这些常用工具整合到一个干净、开源、可自建的平台中？
幸运的是，<strong>IT-Tools</strong> 就是这样一款“开发者的全能工具箱”——功能全面、界面简洁、部署方便，让你不再四处找工具。
<strong>IT-Tools</strong> 是一个开源的在线工具集，专为 IT 从业者与开发者打造。你可以直接在线使用，也可以通过 Docker 一键部署，搭建属于自己的私有工具平台。

GitHub 项目地址：[github.com/CorentinTh/it-tools](https://github.com/CorentinTh/it-tools)

项目在线地址：[it-tools.tech](https://it-tools.tech/)

## <strong>核心特点</strong>

1. 💡 <strong>开源免费，可自建使用</strong>
   代码完全开源，无广告、无登录限制。支持本地或私有服务器部署，保护隐私与数据安全。
2. 🧩 <strong>功能全面，覆盖常用场景</strong>
   从开发、Web、网络到文本处理、加密转换，IT-Tools 几乎包含了所有你能想到的在线工具：

   - <strong>密码编码类</strong>：令牌生成、Hash 文本、RSA 密钥对、加密解密工具。
   - <strong>转换类</strong>：日期、颜色、Base64、YAML/JSON/TOML 转换、进制转换。
   - <strong>Web 类</strong>：URL 编解码、JWT 解析、User-Agent 检测、HTTP 状态码速查。
   - <strong>开发类</strong>：JSON/SQL/XML 格式化、Git 备忘录、Docker 转 Compose。
   - <strong>网络类</strong>：IPv4 子网计算、MAC 查询、IPv6 生成等。
3. ⚙️ <strong>Docker 一键部署</strong>
   只需几行命令，即可通过 Docker 快速部署你自己的 IT-Tools 实例，省心省力。
4. 🎨 <strong>界面简洁，体验流畅</strong>
   无广告、无干扰，所有功能分类清晰、即点即用，极大提升开发效率。
