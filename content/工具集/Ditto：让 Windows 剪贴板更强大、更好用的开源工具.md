## 前言

### 🚀 为什么我推荐用 Ditto，而不是 Windows 自带剪贴板

复制、粘贴、再复制、再粘贴。这是我们每天最频繁的动作之一，但也是最容易忽略的效率环节。

Windows 自带剪贴板虽然方便，但功能非常有限——

🔸 只能保留 25 条历史记录

🔸 重启电脑后就全部清空

🔸 无法分组、搜索或导出

用久了你会发现，它只是“能用”，但远称不上“好用”。

而 Ditto，让复制粘贴这件小事彻底升级。

✨ 无限历史记录（再也不怕复制丢失）

✨ 支持分组、搜索、批量管理

✨ 自动保存，即使重启电脑也能找回所有内容

✨ 还能跨设备同步（同局域网下复制到另一台电脑）

如果用俞军老师的用户体验公式来衡量：
<strong>用户体验 = 新体验 - 旧体验 - 迁移成本</strong>

👉 新体验：高效管理、可搜索、永久保存

👉 旧体验：Windows 剪贴板的功能局限

👉 迁移成本：几乎为零，安装后即可无缝替代

当你第一次在 Ditto 里搜到几天前复制过的一段文字时，

你就会知道——这不是工具的升级，而是效率的质变。
