# 让电脑告别卡顿！分享两个 C 盘清理方法

C 盘就像电脑的 “心脏”，存放着操作系统和重要运行文件。要是 C 盘空间被占满，电脑打开程序会变得慢吞吞，甚至频繁卡顿死机，工作、追剧、打游戏时突然崩溃，简直让人抓狂。定期清理 C 盘，不仅能让电脑 “轻装上阵”，运行如飞，还能避免系统因空间不足出现故障，延长电脑的使用寿命，让它始终保持 “活力满满” 的状态。

以下两个实用的技巧分享给大家。

## 移动“下载”的存储位置

咱们平时下载的文件，默认都会存到 “下载” 文件夹里，可这个文件夹在 C 盘。C 盘是系统盘，存太多东西，空间就不够用啦，电脑容易变慢。所以，赶紧把 “下载” 文件夹挪到其他盘，给 C 盘腾点地儿！

## 磁盘清理

电脑用久了，会产生好多临时文件，就像家里积攒的杂物一样。这些文件一直不清理，会占满 C 盘空间。很多电脑新手不敢随便清理，怕误删重要文件影响电脑运行。别担心，其实磁盘清理没那么复杂，操作对了，既能释放空间，还能让电脑运行更流畅！
