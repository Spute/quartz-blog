## 功能说明

![[FHVPb7370ojJZxx4aPecCXtSn0b.png]]

这款工具最让人惊喜的就是搜索速度，基本一秒就能出结果。不管你电脑里存了多少文件，在它的搜索框里敲入关键词，相关的文件和文件夹立马就出来了，再也不用像等 Windows 自带搜索那样急得抓耳挠腮。

而且它的安装包小得很，才 3MB，安装过程嗖嗖快，也不怎么占电脑内存。现在我在电脑上找资料、文件，基本都靠 Everything，不光快，还支持模糊搜索，能帮我精准找到所有需要的文件，比 Windows 自带的搜索工具好用太多了。

可能有人会问，“Everything” 到底是啥？其实就是 Windows 上的一款搜索引擎，能根据文件名快速找到文件和文件夹在哪儿。和 Windows 自带的搜索不一样，“Everything” 默认会把电脑上所有文件和文件夹都显示出来，就像它的名字一样，能涵盖 “所有东西”，你在搜索框输完关键词，它就会把对应的文件和文件夹筛选出来。

那 “Everything” 索引全部文件要多久呢？因为它只索引文件和文件夹名，所以速度特别快。全新安装的 Windows 10，大概有 12 万个文件，一秒钟就能索引完。就算是索引 100 万个文件，也就大概一分钟，效率是真高。

有人可能还关心它能不能搜文件内容。能的，“Everything” 可以用搜索函数 “content:” 来搜文件内容。不过得注意，要是文件内容没被索引，这时候搜内容就会慢点儿。

担心它占太多电脑资源？完全没必要。“Everything” 特别省资源。全新安装的 Windows 10，大概 12 万个文件，它只需要大概 14MB 的内存和不到 9MB 的硬盘空间。就算有 100 万个文件，也只需要大概 75MB 的内存和 45MB 的硬盘空间，对电脑运行没啥影响。

它还能盯着文件系统的变化，这样搜索结果就能及时跟上文件的变动，保证你找到的都是最新状态的文件。

最让人放心的是，“Everything” 是免费的，而且里面完全没有恶意软件、间谍软件，也没有烦人的广告，用着特别踏实。
