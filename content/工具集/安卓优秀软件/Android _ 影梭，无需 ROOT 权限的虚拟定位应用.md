## 前言

你是否曾因位置暴露而感到隐私受扰？
是否在游戏或社交中因地理限制而错过精彩？
又是否担心被不必要的追踪？

找到一个<strong>开源免费</strong>的应用程序可以让你在没有到达特定位置的情况下，模拟并传送到任何世界上的地理位置，而不必担心您的隐私泄露。

## 功能

影梭是基于 Android 调试 API + 百度地图及定位 SDK 实现的安卓定位修改工具，并且同时实现了一个可以自由控制移动的摇杆。使用影梭，不需要 ROOT 权限就可以随意修改自己的当前位置以及模拟移动。

1. 定位修改
2. 摇杆控制移动，支持模拟步行、跑步、骑行等运动模式
3. 位置搜索
4. 直接输入坐标
5. 历史记录

本软件仅支持：钉钉虚拟打卡、QQ、美团、小红书、陌陌等软件。

经测试并不支持微信、京东等软件。

<strong>务必合规、负责任地使用该软件</strong>
