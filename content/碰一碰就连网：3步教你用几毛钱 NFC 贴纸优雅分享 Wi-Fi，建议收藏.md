# 原理

NFC 贴纸的原理很简单：它就像一张能存储指令的小磁卡。你的手机（需打开 NFC 功能）相当于一个读取器。当手机靠近 NFC 贴纸时，会瞬间读取里面预先写好的信息。

实现“一碰连 Wi-Fi”，需要先用 NFC 信息写入工具，将你家的 Wi-Fi 名称和密码写入一张便宜的 NTAG 芯片贴纸。之后，任何客人用手机一碰这张贴纸，手机就会自动收到指令并连接上这个 Wi-Fi。

PS：特别说明，苹果手机因为系统限制，不支持自动连 WiFi 指令。。。

# 准备工作

淘宝关键字搜 ntag213 或 ntag215，外型是一元硬币大小的圆卡 或 身份证大小的方卡都行，单价一般在 0.5 至 2 元，我买了 10 个 ntag213 也就 4 元。

# 操作步骤

微信小程序搜索「NFC 碰碰卡」，一个完全免费的 NFC 信息写入工具

![[UiWybTeUkoLy0Qx4y5xcCjwBn4b.jpg | 250x500]]

打开后选择「连接 WIFI」，输入家中 WiFi 信息。

![[MjXybjDuEozicGxyDdDcYZpgnyf.png | 250x500]]

将手机靠近 NFC 标签纸，点击「写入 NFC」。即可将 WiFi 账号密码写入到 NFC 标签中。然后把这个贴纸贴到客厅墙上或者桌上。手机碰一碰 NFC 标签纸，就会弹出如下连接 WiFi 提示。

![[ANjbbFQ1do7ugrxPgdsckrMCn4f.jpg | 250x500]]

# 结束语

同理，你也能写入一个网址或 App 链接，实现“一碰即开”的效果。整个过程就是手机读取贴纸里的预设命令并自动执行。比如，制作 NFC 音乐墙，碰一碰听歌；碰一碰投屏看剧，碰一下，手机画面立刻登上大屏。
