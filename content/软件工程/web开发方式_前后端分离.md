# web 开发方式

网站开发方式？

- 前后端分离
- 前后端不分离

两者的区别？前后端分离实现的网站与 django-admin 的网站的区别？

- 浏览器与服务器通讯使用的消息类型不同。前后端分离时使用的是 XHR 和 Fetch（浏览器网络请求 API）获取的动态数据。不分离时，浏览器直接获取 html 文档类型数据。

# 网页渲染方式

方式有？

- SSR（Server Side Rendering）服务端渲染。
- CSR（Client Side Rendering）客户端渲染。

区别是什么?

- SEO 友好：当前绝大多数网站都是中心化的，搜索引擎会从服务端拿到 html。SSR 利于 SEO。
- 首屏加载：SSR 对首次加载速度快。
- 页面切换：CSR 因为只需要重新渲染部分内容，切换更流畅。
- SPA 单页面应用一般采用 CSR。
- MPA 多页面应用一般采用 SSR。

网页渲染是什么？

- 在 web 开发领域，"渲染"是指将代码（通常是 HTML、CSS 和 JavaScript）转换为用户可以在浏览器中看到和交互的可视化界面的过程。包含 html 渲染、css 渲染、布局、绘制、JavaScript 渲染、合成。
