# 告别服务器：用 Cloudflare Pages 免费部署静态网站

## 前言

你是否也曾认为“搭建网站 = 买服务器”？其实，借助 Cloudflare Pages 这类现代托管服务，即使没有后端服务器，也能快速发布一个高速、稳定的静态网站。

我之前和许多初学者一样，对网站托管认知停留在“必须买服务器”的阶段。后来偶然通过朋友了解到 GitHub Pages，才知道静态网站完全可以免费托管。之后同事推荐了 Cloudflare——一个功能丰富且对免费用户极为友好的平台，我才发现原来部署网站可以如此简单。

这让我意识到，很多我们原本以为复杂的技术需求，其实早有更优雅的解决方案。信息差，往往是阻碍我们尝试新工具的最大障碍。

Cloudflare Pages 是一款专业的静态网站托管服务，功能和 GitHub Pages 类似，支持直接部署 HTML、CSS、JS 等静态资源，也可集成 Git 实现自动发布。如果你打算搭建博客、项目介绍页、帮助文档或个人作品集，完全不需要购买服务器。

作为全球顶级的 CDN 服务商，Cloudflare 在全球范围内拥有大量节点，访问速度通常优于 GitHub Pages，国内海外加载表现也更稳定。此外，Cloudflare Pages 允许一个账户创建多个项目，自定义域名，非常适合个人开发者与初创项目使用。

如果你正在寻找一个简单、免费且可靠的静态网站托管平台，不妨试试 Cloudflare Pages。

### 项目主页

项目主页里可以找到，项目访问的链接，如果你没有使用自己的域名，vercel 会给你分配一个子域名。使用这个子域名即可访问你部署的网站。

![[TtKhb0twZo7VdXxoXLLcrAGlnAf.png]]

### 更新 Vercel 部署

在本地修改代码并提交。推送到远程仓库后，Vercel 会 <strong>自动检测到仓库更新</strong>，触发一次新的构建和部署。几十秒后，新版本就会上线。

### 自定义响应头（可选）

对于一些特殊的项目，需要设置自定义的响应头。

Vercel 静态网站需要用配置文件来加自定义响应头。在项目根目录新建 `vercel.json`：

`{ "headers": [ { "source": "/(.*)",  "headers": [ { "key": "Cross-Origin-Opener-Policy", "value": "same-origin" }, { "key": "Cross-Origin-Embedder-Policy", "value": "require-corp" } ] } ] ``}`

解释：

- `source` 是匹配的路径，这里 `/(.*)` 表示所有请求。
- `headers` 就是要追加的响应头。

# <strong>Netlify：国内直连最快、体验最稳的一体化网站托管平台</strong>

## 前言

如果你想拥有一个属于自己的独立博客网站，却又不想花钱买服务器、配置环境、维护运行，那么 <strong>Netlify</strong> 是一个非常理想的选择。
Netlify 是一家位于旧金山的云计算公司，为 Web 应用和静态网站提供托管与无服务（Serverless）后端服务。它能自动从 Git 仓库拉取代码、构建并部署网站，支持<strong>全球 CDN 分发</strong>，让你可以用极低的门槛发布并维护自己的个人博客。

简单来说，只要你有一个 GitHub 账号，就能通过几次点击，把你的 Jekyll、Hugo 或 Hexo 博客一键部署上线。

## 特点

1. <strong>免费且高效的部署服务：</strong>Netlify 提供的免费套餐足以满足个人博客需求：

   - 每月 <strong>100GB 免费流量</strong>
   - <strong>300 分钟构建时间</strong>
   - <strong>1 个同时编译队列</strong>对于普通博客来说，这些资源已经绰绰有余。一个静态博客通常构建不到一分钟，即便每月频繁更新，也完全能覆盖。
2. <strong>一键从 GitHub 部署：</strong>Netlify 与 GitHub 深度集成。你只需授权登录，即可自动拉取代码、编译并发布网站。每次推送新内容，Netlify 都会自动重新构建并上线，无需人工干预。
3. <strong>全球加速与安全防护：</strong>得益于 Netlify Edge 的全球 CDN 网络，访问速度快且稳定。平台自带流量清洗功能，即便遇到异常访问或攻击，也能有效防护，不会造成流量暴涨。
4. <strong>完整 HTTPS 支持：</strong>Netlify 与 Let’s Encrypt 完全集成，网站会自动配置 HTTPS，无需手动申请或续期证书，让你的网站安全又省心。
5. <strong>无服务功能扩展：</strong>除了静态托管外，Netlify 还支持无服务表单、身份验证以及 AWS Lambda 函数等功能，让博客有更多交互和动态能力。
6. <strong>成本低、维护轻：</strong>个人使用完全免费，无需服务器运维知识。再配合图床托管策略，可以显著降低带宽消耗、提升加载速度。
