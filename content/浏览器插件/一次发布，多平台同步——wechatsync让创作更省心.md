## 前言

写文章的人都知道，把同一篇内容同步到多个平台是一件又累又烦的事——不仅要重复排版，还要一遍遍上传。
如果你也为一次编辑、N 个平台的重复操作头疼，可以试试一个小工具：<strong>wechatsync</strong> 浏览器插件。
它支持微信公众号、今日头条、WordPress、知乎、简书、掘金、CSDN、typecho 等平台，一次发布，就能自动同步到多个平台，省时省力，专心创作。

- github 项目地址：[https://github.com/wechatsync/Wechatsync/tree/master](https://github.com/wechatsync/Wechatsync/tree/master)
- 工具页面：[https://www.wechatsync.com/?utm_source=github_info](https://www.wechatsync.com/?utm_source=github_info)

## 安装和使用

说明：目前该项目已经停止维护，chrome 浏览器增加限制，目前只支持 v3 版本插件，因此 chrome 浏览器已无正常使用，需要使用微软的 Edge 浏览器才支持 v2 版本的插件

### 同步

登录在线工具：[https://www.wechatsync.com/md/#/](https://www.wechatsync.com/md/#/)

编辑好内容后，在 markdown 编辑器中点击发布（需要登录相关平台）

选择平台，点击确认后，自动同步到平台的草稿中，点击查看草稿链接，即可跳转。
