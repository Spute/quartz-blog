# 扣子版 Manus 免费内测，扣子空间抢先体验！

## 是什么

官方介绍：扣子空间是你和 AI Agent 协同办公的最佳场所。在这里，精通各项技能的通用实习生，各行各业的“领域专家”，任你选择。把任务交给扣子空间，把时间还给自己。

个人理解：扣子空间的探索模式和 manus 非常类似，是完全由 AI 自主决策规划任务。规划模式则是帮你规划分解任务，同时支持修改，可以人为干预任务的执行，这点有点像 flowith 的 Oracle 模式。

官方地址：[https://www.coze.cn/space-preview](https://www.coze.cn/space-preview)

## <strong>扣子空间有什么特点？</strong>

<strong>🌟 从回答问题，到解决问题，让 Agent 帮你完成更多的工作</strong>

- 自动分析需求，拆解为多个子任务
- 自主调用工具（浏览器、代码编辑器等），执行任务
- 输出完整的结果报告，例如网页、PPT 、飞书文档等

<strong>🌟 专家 Agent 生态，让更专业 Agent 来为你提供服务</strong>

- 「华泰 A 股观察助手」可以为你进行每日早报生成；针对股票分析问题，助手也能可以为你答疑解惑
- 「用户研究专家 」可以协助你进行用研资料深度分析，省时省力地助你获取更多用户洞察

<strong>🌟 探索/规划双模式，更好地和 Agent 一起协作完成高难度任务</strong>

- 探索模式：让 AI 自主动态探索，完成速度更快
- 规划模式：让 AI 深度思考，适合高复杂性任务

<strong>🌟MCP 扩展集成，无限拓展 Agent 能力边界</strong>

- 首批官方支持飞书多维表格、高德地图、图像工具、语音合成等 MCP
- 即将支持「扣子开发平台」发布 MCP 至「扣子空间」

## 探索模式测试

为了对比，我用之前测试 manus 相同的任务“制作一个面向小白的 three.js 课程”，以下是两者的回放。

manus 的回放：https://manus.im/share/YR0QgrlAxo9oUXufSWcFdv?replay=1

扣子空间的回放：[https://space.coze.cn/s/a9joCzQbT3Y/](https://space.coze.cn/s/a9joCzQbT3Y/)

个人感受是，扣子空间的回答会偷懒，首次提问只生成了课程大纲就停了，之后我让他继续生成课程内容，也只是在原有 md 文档基础上加了一些代码案例，但代码不够完整，且不支持预览效果，于是我让他生成对应的静态网站，这次花了十多分钟，生成了一个 jsx 文件，打开后也显示不了，看后台显示是有报错的。对比下来感觉扣子空间的能力会比 Manus 差许多，还有待提高，不过目前它还处在内测阶段，多点给点鼓励和期待吧。
